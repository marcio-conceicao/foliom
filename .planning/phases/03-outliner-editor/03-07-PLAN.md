<interfaces>
From plans 03-03 / 03-05 / 03-06: the full mutation API surface:
- `PUT /api/blocks/:id` — edit raw
- `POST /api/blocks` — insert sibling / child
- `PATCH /api/blocks/:id/structure` — indent/outdent/move
- `DELETE /api/blocks/:id` — delete / merge
- `POST /api/pages` — create page
- `POST /api/pages/:name/rename` — rename + rewrite backlinks
- `GET /api/autocomplete` — passive; not exercised here

ACPT-01 round-trip test pattern (from Phase 1):
- For each .md file in a directory: read bytes → `segment(bytes)` → for each block, slice `bytes[byte_offset..byte_offset+byte_length]` → concatenate → assert equals input bytes.

Phase 1 synthetic corpus: `crates/core/tests/fixtures/logseq-synthetic/` (10 fixture .md files + README sibling, 11 total). This plan curates a subset for ACPT-05 — see Task 1 action.
</interfaces>

    #[tokio::test(flavor = "current_thread")]
    async fn acpt_05_scripted_edit_sequence() {
        let (td, state, app) = setup();
        // Step 1: edit
        // Step 2: insert sibling
        // ... etc ...

        // Byte invariants
        assert_no_crlf_introduced(&td);
        assert_no_bom_introduced(&td);

        // Foliom-metadata grep
        for path in walk_md(&td) {
            let bytes = std::fs::read(&path).unwrap();

            }
        }

        for path in walk_md(&td) {
            let bytes = std::fs::read(&path).unwrap();
            let s = std::str::from_utf8(&bytes).unwrap();
            let _ = pulldown_cmark::Parser::new(s).count(); // no panic
        }

        // ACPT-01 corpus replay
        for path in walk_md(&td) {
            let bytes = std::fs::read(&path).unwrap();
            let blocks = foliom_core::parser::segment::segment(&bytes);
            let mut reconstructed = Vec::with_capacity(bytes.len());
            for block in &blocks {
                reconstructed.extend_from_slice(&bytes[block.byte_offset..block.byte_offset + block.byte_length]);
            }
            assert_eq!(reconstructed, bytes, "ACPT-01 round-trip failed for {}", path.display());
        }
    }
