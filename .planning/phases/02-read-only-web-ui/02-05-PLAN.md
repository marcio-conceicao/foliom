      import { fetchBacklinks, type Backlink } from '$lib/api';

      let { name }: { name: string } = $props();
      let backlinks = $state<Backlink[] | null>(null);

|----------|-------------|
| Backlink snippet text → DOM | Always textContent (never `{@html}`) to avoid XSS from raw backend strings |
| `sidebarPages` set lookup → chip styling | Derived `Set<string>` from store; no user input crosses here |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-17 | XSS | Backlink snippet | mitigate | Use Svelte textContent binding (`{text}`) not `{@html text}` for snippets. Snippets contain stripped raw markdown — never trust as HTML. |
| T-02-18 | XSS | Sidebar page names | mitigate | Page names come from the backend (canonical, validated by Phase 1 path normalization). Render via textContent binding; URL building uses `encodeURIComponent`. |
| T-02-19 | Info disclosure | Theme localStorage | accept | Theme preference is non-sensitive. |
